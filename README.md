# Skillforge App 📚

Android course learning app built for Clickretina assessment.

## Screens
- Home — Popular courses + Categories
- Course Detail — Info, instructor, lessons, enroll button  
- Video Player — ExoPlayer, seekbar, lesson list

## Tech Stack
- Kotlin, MVVM
- Retrofit + Gson (API)
- ExoPlayer Media3 (video)
- ViewBinding, RecyclerView
- LiveData + ViewModel

## API
https://raw.githubusercontent.com/android-assesment/notes/refs/heads/main/data.json

## How I Used AI (Claude)

- Built Retrofit, Repository, ViewModel boilerplate
- Debugged ExoPlayer 403 errors and RecyclerView issues

## Setup
1. Clone repo
2. Open in Android Studio
3. Run on device (API 24+)
